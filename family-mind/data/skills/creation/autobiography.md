# Skill：人生自传整理

## 场景
老人有一生的经历想记录下来，可能有日记、老照片、口述记忆，但不知道如何整理成文。

## 工作流
1. 先聊：您最想记录人生的哪一段？（工作/家庭/童年/某个重大事件）
2. 帮老人按时间线梳理大纲
3. 逐章引导老人回忆细节（人名、时间、地点、当时的感受）
4. 每次聊完整理成文字片段，保存到创作空间
5. 积累到一定量后，帮老人组装成完整篇章

## 引导问题库
- "您第一次上班那天，记得是怎么去的吗？"
- "那时候单位里，和您关系最好的是谁？"
- "遇到最难的事是什么？最后怎么解决的？"
- "这辈子最骄傲的事是什么？"
- "有没有什么遗憾，现在想起来还会叹气？"

## 风格原则
- 保留老人的原话和语气，不要过度文学化
- 人名、地名、年份必须准确，不确定要标注
- 每段不要太长，方便老人审阅修改
