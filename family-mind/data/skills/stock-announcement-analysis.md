# A股公告分析 Skill

## When to Use

当用户要求分析以下类型的A股公告时：
- 期货套保公告
- 产能扩张公告
- 海外投资公告
- 重大合同公告
- 其他需要深度分析的财经公告

## Procedure

### 1. 公告识别与分类
- 确定公告类型
- 提取关键信息（公司、金额、时间、标的）

### 2. 主动性关联分析（根据公告类型）

**期货套保公告 → 查：**
- 相关商品价格走势
- 地缘政治影响
- 行业周期位置

**产能扩张公告 → 查：**
- 行业供需情况
- 价格预测
- 竞争格局变化

**海外投资公告 → 查：**
- 地缘政治风险
- 汇率风险
- 当地政策环境

**重大合同公告 → 查：**
- 客户资质
- 行业地位
- 履约能力

### 3. 信源可靠性分级

**Tier 1 - 优先使用：**
- 卓创资讯、隆众资讯
- 高盛、中金等研报

**Tier 2 - 权威媒体：**
- Reuters、Bloomberg
- 财新、证券时报

**Tier 3 - 行业媒体：**
- 期货日报、化工在线

**谨慎使用：**
- 自媒体、论坛、未经证实的消息

### 4. 交叉验证
- 至少2-3个独立信源验证关键数据
- 数据矛盾时说明不确定性
- 明确标注信息来源和发布时间

### 5. 输出格式

```markdown
## 公告要点
...

## 关联分析
...

## 数据来源
- [来源1](链接) - 发布时间
- [来源2](链接) - 发布时间

## 风险提示
...
```

## Pitfalls

- **不要**仅依赖单一信源
- **不要**忽略公告中的风险提示条款
- **不要**过度解读未经证实的市场传闻
- **注意**区分公司公告原文和市场解读

## Verification

分析完成后检查：
- [ ] 是否覆盖了该公告类型的关键分析维度？
- [ ] 是否使用了至少2个独立信源？
- [ ] 是否标注了所有数据来源？
- [ ] 是否说明了数据矛盾或不确定性？
