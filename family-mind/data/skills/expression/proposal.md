# Skill：意见整理成方案

## 场景
老人在业委会、单位联谊、社区活动中有想法，但说不系统，想整理成一份正式的方案或建议。

## 工作流
1. 让老人随便说：您想解决什么问题？
2. 逐条记录老人的意见（不打断，不评判）
3. 帮老人分类：事实 / 问题 / 建议 / 请求
4. 整理成结构化文档：背景、现状、问题、建议措施、预期效果
5. 生成几个版本：正式版（给组织）+ 口语版（老人自己讲）

## 示例
老人说："我们小区停车位不够，晚上回来没地方停，有些人还乱停挡路，物业也不管。"
→ 整理成：
- 背景：小区 X 栋楼，Y 户，现有车位 Z 个
- 问题：缺口 XX 个，晚 8 点后无位，消防通道被占
- 建议：1) 重新划线 2) 外来车辆限时 3) 安装地锁
