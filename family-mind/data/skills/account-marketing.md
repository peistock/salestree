# 客户营销触点研究

## 目标

在已经了解公司背景的基础上，单独研究它的营销打法。回答：这家公司怎么触达用户、在哪里投广告、内容调性是什么、和谁合作、campaign 节奏如何。

## 执行流程

1. **识别公司**：从用户输入中提取公司名；如果不确定，先反问确认。
2. **加载已有研究**：先用 `get_account` 查 CRM，如果已有 `research_summary`，先了解公司背景，避免脱离业务空谈营销。
3. **并行收集外部信息**：用 `search_web` / `jina_reader` / `browse_open` 收集：
   - **品牌定位**：品牌口号、视觉风格、目标人群画像。
   - **线上触点**：官网、公众号/视频号、微博、抖音、小红书、B 站、天猫/京东旗舰店等。
   - **内容策略**：主要内容形式（短视频、图文、直播、白皮书）、调性、更新频率。
   - **投放分析**：信息流广告、搜索广告、开屏/贴片、达人种草、KOL/KOC 合作。
   - **Campaign 节奏**：近 6-12 个月重大营销节点、联名/IP、话题营销。
   - **私域运营**：企微、社群、会员体系、裂变玩法。
   - **竞品营销对比**：主要竞品的营销差异。
4. **整理输出**：按下面格式输出 Markdown，**每个事实必须标注来源**。
5. **收尾动作**：输出后询问是否需要：
   - 生成针对该公司的营销方案（转 marketing-plan）
   - 深挖某个渠道/campaign
   - 生成跟进文案（转 outreach-drafter）

## 输出格式

```markdown
# 【公司名】营销触点分析

## 一、品牌定位
- 目标人群画像
- 品牌口号/价值主张
- 视觉与调性
- 来源：[标题](URL)

## 二、线上触点分布
| 平台 | 账号/店铺 | 内容类型 | 更新频率/粉丝量级 |
|------|----------|---------|------------------|
| 抖音 | … | 短视频/直播 | … |
| 小红书 | … | 种草笔记 | … |
| 公众号 | … | 品牌内容/活动 | … |
| 天猫/京东 | … | 电商详情页 | … |
| 官网 | … | 品牌/产品 | … |

## 三、内容策略
- 主要内容形式
- 内容调性
- 爆款内容特征
- 来源：[标题](URL)

## 四、投放与合作
- 信息流/搜索广告观察
- KOL/KOC 合作特征
- 达人种草方向
- 来源：[标题](URL)

## 五、Campaign 节奏
- 近 6-12 个月重大营销节点
- 联名/IP/话题营销
- 大促参与度
- 来源：[标题](URL)

## 六、私域与会员
- 企微/社群/会员体系
- 裂变与复购玩法
- 来源：[标题](URL)

## 七、竞品营销对比
- 主要竞品 2-3 家，每家一句话营销差异
- 来源：[标题](URL)

## 八、机会观察
- 可借鉴的打法
- 可切入的空白点（如需输出完整方案，转 marketing-plan）
```

## 铁律

- **不带入销售预设**：先描述客观事实，不急着说「我们可以做什么」。
- **必须带来源**：每条信息标注来源。
- **结合业务背景**：营销打法必须和公司主营业务、目标人群联系起来，不要孤立罗列。
- **控制长度**：不超过 1500 字。
