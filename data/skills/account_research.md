# 客户公司深度研究

## 目标
帮助销售人员在见客户前快速掌握客户公司的关键信息，从"介绍产品"升级为"诊断业务"。

## 研究维度

### 1. 公司基本面
- 公司简介、成立时间、总部地点
- 行业地位、市场份额
- 主营业务、产品线
- 融资历程、估值

### 2. 财务与经营
- 营收规模、增长率
- 盈利能力
- 主要客户/合作伙伴
- 最新财报关键数据

### 3. 组织与人事
- 高管团队及背景
- 组织架构变动
- 关键岗位招聘动向
- 企业文化特点

### 4. 业务动态
- 最新战略方向
- 新产品/服务发布
- 市场扩张计划
- 数字化转型进展

### 5. 竞品与行业
- 主要竞争对手
- 行业趋势影响
- SWOT 分析

### 6. 数字化营销现状
- 现有广告投放渠道
- 营销技术栈
- 社交媒体运营情况
- 电商业务布局

## 执行流程
1. 确认客户公司名称
2. 使用 search_web 搜索最新信息
3. 使用 search_industry_news 搜索行业资讯
4. 使用 get_account 查询 CRM 已有数据
5. 整合信息，生成结构化研究报告
6. 标注信息来源

## 输出格式
```markdown
# 客户研究：【公司名称】

## 公司概况
...

## 最新动态
...

## 关键联系人
...

...

## 建议切入点
...
```
